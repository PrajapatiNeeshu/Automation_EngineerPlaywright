# 4ï¸âƒ£ normalize-space() â€“ bahut IMPORTANT interview topic

### Problem

HTML me extra spaces, newline aa jate hain:

```html

   Email   
</label>
```

### Ye fail karega âŒ

```xpath

```

### Correct way âœ…

```xpath

```

### normalize-space kya karta hai?

* Leading space hata deta hai
* Trailing space hata deta hai
* Beech ke multiple spaces ko single bana deta hai

### Contains + normalize-space

```xpath

```

ğŸ‘‰ **Text locator likhte waqt ALWAYS normalize-space use karo**

# 7ï¸âƒ£ Interview me kaise explain kare (ready-made answer)

## 1ï¸âƒ£ Git kya hai?

**Answer:**
Git ek **distributed version control system** hai jo source code ke changes ko track karta hai. Isse multiple developers ek hi project par saath kaam kar sakte hain bina ek-dusre ka code overwrite kiye.

---

## 2ï¸âƒ£ Git aur GitHub me difference kya hai?

**Answer:**

| Git                        | GitHub                       |
| -------------------------- | ---------------------------- |
| Local tool                 | Cloud platform               |
| Version control karta hai  | Code hosting + collaboration |
| Offline bhi kaam karta hai | Internet required            |

ğŸ‘‰ Git = tool, GitHub = service

---

## 3ï¸âƒ£ Repository kya hoti hai?

**Answer:**

* Code
* Git history
* Branches
  sab store hote hain.

## 4ï¸âƒ£ `git clone` aur `git pull` me difference?

**Answer:**

* `git clone` â†’ pehli baar repo copy karta hai
* `git pull` â†’ existing repo ko update karta hai

## 5ï¸âƒ£ Branch kya hoti hai?

## 7ï¸âƒ£ Git conflict kya hota hai?
